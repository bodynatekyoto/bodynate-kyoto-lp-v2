## 会話の流れ

### ① ファーストビュー改善 — 3円ポイントの位置移動
**依頼内容：**  
「3つの丸ポイントをファーストビューで見えるように写真直下に移動してほしい」

**実装内容：**  
- `.fv-circles` を `.fv-content` の最初の子要素に移動（divider・メッセージより上）
- `fv-content` の `padding-top` を 40px → 24px に縮小
- circles の `padding-top` も調整
- スマホ画面でスクロールせずに3ポイントが見えるよう配置

---

### ② FAQアコーディオン実装
**依頼内容：**  
FAQをクリックで開閉できるアコーディオン形式にしてほしい

**実装内容：**
- CSS: `.faq-a { display: none; }` / `.faq-item.open .faq-a { display: block; }`
- トグルアイコン（＋）を丸型ボタンで実装、open時に45度回転してゴールドに変色
- JS: クリックで `.open` クラスを付け外し（他は全部閉じてから対象を開く）
- デフォルトで1問目のみ開いた状態

---

### ③ スクロールリビール（フェードイン）
**依頼内容：**  
スクロールしたらセクションがふわっと表示されるようにしてほしい

**実装内容：**
- `Intersection Observer API` を使用（JS）
- `.reveal` クラスを12セクションに付与
- CSS: `opacity: 0; transform: translateY(30px)` → `.visible` で `opacity: 1; translateY(0)`
- `threshold: 0.08 / rootMargin: 0px 0px -40px 0px` の設定

---

### ④ BAセクション Before画像グレースケール
**依頼内容：**  
ビフォーアフターのBefore側の画像をモノクロっぽくしてほしい

**実装内容：**
```css
.ba-side:first-child img {
  filter: grayscale(0.75) brightness(0.88);
}
```

---

### ⑤ 残り枠バー（緊急性演出）
**依頼内容：**  
「残り〇名」の予約枠バーをCTAの近くに入れてほしい

**実装内容：**
- プログレスバー形式（`.fv-slots-bar` / `.fv-slots-fill`）
- 「今月の残り予約枠 **残り3名**」のテキスト付き
- ゴールドCTAボタンの上に配置（緊急性が伝わる位置）
- アニメーション付き（slots-pulse）

---

### ⑥ プログラム月別コネクター矢印
**依頼内容：**  
3ヶ月プログラムのカード間に矢印ラインを入れてほしい

**実装内容：**
```css
.program-connector { display: flex; align-items: center; height: 36px; position: relative; }
.program-connector::before { /* 縦ライン（グリーン→ゴールドのグラデーション） */ }
.program-connector::after { content: '▼'; /* 三角アイコン */ }
```

---

### ⑦ 価格セクション 緊急バッジの修正
**問題：**  
`.pricing-card` に `overflow: hidden` があるため、`::before` で作ったバッジ（`top: -14px`）が切れていた

**解決策：**  
- `::before` 疑似要素をやめて、実際のHTML要素 `.pricing-hot-badge` に変更
- `.pricing-trial-wrap` でカードをラップ、バッジはラップ側に配置

```html
<div class="pricing-trial-wrap">
  <div class="pricing-hot-badge">🔥 今すぐ予約できます</div>
  <div class="pricing-card pricing-trial">...</div>
</div>
```

---

### ⑧ スティッキーCTA（固定ボタン）
**依頼内容：**  
スクロールしたら下部にCTAボタンが固定表示されるようにしてほしい

**実装内容：**
- `position: fixed; bottom: 0;` のスティッキーCTA
- スクロール200px以上で表示、200px未満で非表示（JS制御）
- `passive: true` でパフォーマンス最適化

---

### ⑨ アクセスセクションの構成変更

**依頼内容：**  
「他の店舗の紹介は一番最後にして下さい」

**変更内容：**
- 中盤のアクセスセクション → **神戸三宮店のみ**に変更
- 京都店・和歌山店 → **一番最後（最終CTAの後）** に移動

**デザインについて：**  
最初は別デザイン（`.other-stores`）で実装したところ  
「**デザイン変えないで**」と指摘。  
→ 同じ `section.access` クラスをそのまま使用して統一
